פעולה קובץ מקור קובץ יעד / שירות כיוון נתונים שעוברים
כניסה עם Google Login.tsx → handleGoogle() Supabase Auth → Google OAuth OUT → IN redirectTo URL → JWT session token
חזרה מ-OAuth AuthCallback.tsx supabase.auth.getSession() POLL ?code= param → session object
בדיקת הרשאות useAuth.ts Supabase: profiles table OUT → IN user.id → { is_admin: bool }
שמירת brief useBriefs.ts → saveBrief() Supabase: briefs table OUT { id, state: WizardState (JSONB), title, cluster_id }
טעינת briefs useBriefs.ts → fetchBriefs() Supabase: briefs table IN BriefRecord[] ordered by updated_at
חישוב TAKAM calc.ts → calcRoleMonthlyCost() — LOCAL MixEntry + rolesData + hoursMultiplier → ₪ עלות
AI Advisor AiAdvisorModal.tsx → briefAIHelper.ts Anthropic API (claude-*) OUT → IN CalcState JSON → המלצות טקסט
ייצוא Word wordExport.ts — LOCAL WizardState → .docx blob → download
ייצוא PDF Step4Results.tsx html2pdf.js (client-side) LOCAL DOM → PDF blob → download
